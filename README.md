# old-cs-projects
Old CS Projects

Most of these projects use custom data structures

## AustraliaMap:
Enter the abbreviation of the start and end location of the desired trip. Uses Dijkstraâ€™s algorithm to find the shortest path.

## Blackjack:
Press hint to hear a voice that gives generic advice. Press cheat to hear a voice that gives advice based on the top card of the deck.

## Bounce:
Bouncy ball physics simulator. Grab and throw the ball using the mouse. Press space to randomize the ball size and gravity. When weird gravity is turned on, the compass needle in the middle of the screen points in the direction of the gravity with its length proportional to the strength of the gravity.

## EgyptianWar:
Multi-player Egyptian War game. Run the Server first and then the Client. Replace "localhost" with the Server IP Address in ClientScreen.java when using separate devices.

## PixelArt:
Drawings can be saved as a png and later reuploaded. 

## SpaceShooter:
Single-player Space Invaders style game. Use arrow keys to move and space to shoot. See tutorial in game. Press 'o' on keyboard to skip levels. Press escape to skip to end screen. 'z', 'x', 'c', and 'v' are cheat keys to get upgrades.

## SpellBound:
Animated 2 player boardgame on 1 device. See tutorial in game.

## USTourist:
Explore the US as a tourist. Press space to make the map in the top left appear and disappear. Go near a landmark to see its description.

## Warehouse:
When gravity is turned on, objects are moveable and stackable. Interact with objects using the mouse. Grab objects from the right side of the screen or spawn them using the buttons on the bottom.
